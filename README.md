Kiddopia is a popular educational gaming app for young children that offers interactive learning activities.Today, Kiddopia is used by families worldwide and generates most of its revenue from the United States market, with millions of downloads and a strong subscription-based model.generates most of its revenue from the United States market,l.
. Project Title

This project analyzes the organic search performance of the Kiddopia website using data from Google Search Console. The goal of the project is to visualize key SEO metrics and identify patterns in user engagement and search visibility.
Using Tableau, the dataset was transformed into an interactive dashboard that helps understand how users interact with the website through Google search results.
The analysis focuses on identifying traffic trends, engagement levels, and ranking performance to support data-driven SEO decisions.
Dataset

Source: Google Search Console
The dataset contains search performance data including:
Clicks – Number of times users clicked the website from search results
Impressions – Number of times the website appeared in search results
CTR (Click Through Rate) – Percentage of impressions that resulted in clicks
Average Position – Average ranking of the website in search results
Time – Hourly performance data
This data helps evaluate the website’s visibility and engagement in organic search.
Dashboard Visualizations

The Tableau dashboard includes the following visualizations:
1. Clicks Trend
Shows how many users clicked the website across different hours of the day.
2. Impressions Trend
Tracks how frequently the website appears in Google search results.
3 CTR Trend
Shows the percentage of impressions that converted into clicks, helping evaluate search snippet effectiveness.
4. Average Position Trend
Displays the average ranking position of the website in search results over time.
5. Clicks vs Impressions Analysis
A scatter visualization showing the relationship between search visibility and user engagement.

Key Insights
Website traffic varies throughout the day, indicating peak user activity hours.
High impressions do not always lead to higher clicks, highlighting opportunities to improve search result titles and descriptions.
CTR fluctuations suggest potential improvements in search result optimization.
Ranking position trends help identify periods of stronger search performance.
